# Project 2

A simple web app that keep track of houses visited by a buyer

## technologies Used
- Invision (wireframe)
- lucy.app (ERD)
- Trello Board (https://trello.com/b/leSYWMqE/project-2)
- HTML5
- CSS3
- Node.js
- Express
- MongoDB 
- Github Pages (for saving)
- Heroku Pages (for depolyment)
